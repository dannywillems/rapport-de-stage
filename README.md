# rapport-de-stage

Rapport de stage de master 2.

## Compiler les fichiers TeX.

Certains fonts sont nécessaires. `tlmgr` peut être utilisé pour les installer:
```
tlmgr install collection-fontsrecommended
```

Si vous utilisez `apt-get` ou `aptitude`, vous pouvez utiliser:
```
sudo apt-get install install texlive-fonts-extra
```
